# SQL-DATA-WAREHOUSE-PROJECT-

This project provides a clear demonstration of an end-to-end data warehousing and analytics solution built on SQL Server. It includes ETL process implementation, structured data modeling, and analytics-ready data transformation. The data warehouse is designed following the Medallion Architecture, separating raw, cleansed, and curated data across Bronze, Silver, and Gold layers to ensure scalability, data quality, and efficient analytics.

-----------------------------------------------------------------------------------------------------------------

## Project Rrequirements

### Data Warehouse (Data Engineering project)

#### Objective
The objective of this project is to design and develop a centralized data warehouse using SQL Server to consolidate sales data from multiple sources. The solution aims to enable reliable analytical reporting and support data-driven decision-making for business development by providing accurate, timely, and structured insights into sales performance.
